Repository erstellen
git init bierstile

Datei erstellen
nano bierstile.yaml

Datei zur stage hinzufügen
git add bierstile.yaml

Git sagen wer man ist
  git config --global user.email "you@example.com"
  git config --global user.name "Your Name"
  
Commiten
git commit -m "Nachricht die die Änderung beschreibt"

Änderungen anzeigen
git log
